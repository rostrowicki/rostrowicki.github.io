Minikube Cheatsheet

Rob published on 2022-07-26
Minikube is a tool that allows you to run a single-node Kubernetes cluster locally on your machine. It is a great way to learn and experiment with Kubernetes without the need for a large, complex cluster. One of the key concepts of Minikube is the virtual machine (VM) that it runs on. Minikube uses a VM to create a self-contained environment in which to run the Kubernetes cluster. By default, Minikube uses the VirtualBox hypervisor to create the VM, but it can also use other hypervisors such as VMware Fusion and Hyper-V.
